# AI Drone Detection using YOLOv8

## Overview

This project implements a custom drone detection system using YOLOv8 and Computer Vision.

## Features

* Custom-trained YOLOv8 model
* Drone detection from images
* Image upload testing pipeline
* Google Colab implementation

## Technologies Used

* Python
* YOLOv8
* Ultralytics
* PyTorch
* OpenCV
* Google Colab

## Dataset

AOD Drone Dataset

## Training Configuration

* Model: YOLOv8n
* Epochs: 80
* Image Size: 640

## Results

Successfully detected drones from aerial imagery using a custom-trained model.
